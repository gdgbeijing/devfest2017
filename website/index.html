Devfest 2017·北京站

北京 GDG 年度开发者大会

视频回放 下载 PPT

  • DevFest 是 Google 每年秋季在全球 GDG（谷歌开发者社区）推行的，针对 Google 技术开发者、爱好者所举办的技术交流活动。其主要目的在于同步 Google 相关各种先进技术，并进行推广与应用，促进本地社区成员的交流。
  • DevFest活动除设置有吸引力的主题分享，还有互动参与环节和来自 Google 的正版周边礼物，对于 Google 技术开发者、爱好者都是不可错过的体验机会。
  • DevFest 2017 北京站由北京谷歌开发者社区主办，北京邮电大学协办。我们非常荣幸邀请到了来自海内外数十位资深工程师，除了 Google 的研发专家团队，还非常荣幸地邀请到了 JetBrains 团队，这也是 JetBrains 团队在中国大陆首次针对 Kotlin 的官方分享。另外，我们还邀请到了国内诸多一线研发团队，滴滴、Strikingly、百度、集智社区等公司，带来他们近期的一线研发经验。

Android 讲师

《When Wireless Communications Meet Artificial Intelligence》
北京邮电大学教授

彭木根

为了更好的满足人与物、物与物通信，无线通信网络需要借助人工智能技术提升组网性能及减少人工投入成本等。本演讲将介绍无线网络发展面临的挑战，重点描述人工智能学习算法如何应用于无线网络，以及基于人工智能算法的无线网络研究进展。

《What's new in Android O》
Google 开发技术推广工程师

高寒蕊

主要介绍需要开发者注意的Android O平台的一些变化，以及会对现在的app造成的影响。

《Kotlin from zero to how can it help me?》
JetBrains 开发者推广团队

Hadi Hariri

在本演讲中我们将从基础出发，侧重在通过既有的Java代码来了解Kotlin的优势以及Kotlin能够如何帮助我们的日常工作。我们将从Kotlin语言的基础知识一路延伸到函数式结构特性，以及它如何能创造更干净简洁的代码。

《Introduction to Google ARCore 》
Google 研发工程师

Jack Fan

谷歌经过在AR领域多年的研发积累在今年8月发布了Google ARCore的预览版，目标将支持上亿个安卓设备。这里将介绍ARCore的功能以及对AR领域未来的展望。

《滴滴插件化平台VirtualApk》
滴滴高级研发工程师

郑涛

主要介绍围绕滴滴乘客端业务，滴滴首个开源项目，VirtualAPK的演进历程，及插件引擎原理和插件构建的技术细节。希望能让开发者对Android系统及Gradle有更深入的了解。

《Kotlin as Your Next Language》
Strikingly 联合创始人 & CTO

郭达峰

将和大家分享我们公司对目前多门主流语言进行分析，从编程风格、社区生态圈、工具链、类型检查几个维度深入分析和探讨，最终确定纳入 Kotlin 作为公司新的一门服务端语言技术栈。本次分享将会带领大家纵览主流语言的几个特性和风格，相信对于编程语言有兴趣但还在徘徊犹豫的开发者会有一些启发。

《Android Things系统结构及展望》
Google IoT Developers Experts谷歌开发技术专家

王玉成

Android Things的功能，整体框架，以及软件开发调试环境介绍。Google IoT Core介绍，以及现有的基于Android Things的项目概览

Web 讲师

《Understanding Modern Web Development》
资深前端工程师

杨扬

三个新时代给 App 和 Web 带来的变革和趋势、新分工和『现代 web 开发』等话题

《从 UI 到 AI —— 移动端 H5 页面生产技术漫谈》
百度高级研发工程师

潘征

第一个基于 css 编写的圆角按钮，宣告网页进入绚丽多彩的新时代。从 html5 到 css 动画，到 canvas，SVG，WebGL，VR，AR；从纯手工编写，到各种封装库，到拖拽生成，再到 AI 辅助设计。移动 H5 页面资深玩家带你领略这些年手机屏幕 Web 交互的巨变。

《Angular5功能简介与现代Web App设计》
jizhi.im CTO

魏钊旸

从最新发布的Angular5的各个特点出发，说明当前前端框架发展趋势以及Web App设计中设计师应当注意的种种问题，并从实践角度出发，介绍如何使用Angular在实际项目中高效、完美地解决这些问题。

TensorFlow 讲师

《Effective Tensorflow for non-experts》
Google 研发工程师

廖宝华

同时介绍estimator和keras API，TensorFlow team推荐使用的high level API。

《TensorFlow 实现 CapsNet》
OpenBayes 机器智能实验室

姜汉

讲解 Capsule 和 Neuron 的区别，对比 CapsNet 和 CNN 在实践过程中的效果。使用 TensorFlow 实现 CapsNet 并与实际项目数据进行对比。

《TensorFlow and Probabilistic Programming》
集智社区 联合创始人&CEO

王文凯

1. 介绍什么是“概率编程”，为什么重要；2. TensorFlow不仅能用于神经网络，也能用于概率编程；3. 现场演示一个基于TF的概率编程实例
